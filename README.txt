DESCRIPCIÓN:
  Módulo de python llamado 'imgops' con varias operaciones de procesado de imagen:
    adjustIntensity
	  equalizeIntensity
	  filterImage
	  gaussKernel1D
	  gaussianFilter
	  medianFilter
	  highBoost
	  erode
	  dilate
	  opening
	  closing
	  fill
	  gradientImage
	  edgeCanny
	  cornerHarris
  Puede usar la función built-in 'help' para obtener más información sobre estas.

USO:
  Para importar este módulo, incluya la carpeta 'imgops' en su carpeta de 'src'
  y así poder hacer 'import src.imgops' en otros módulos de esa carpeta. 

TEST:
  Puede probar cada función del módulo con 'python -m src.imgops FUNC...' donde
  'FUNC' es el nombre de la función a probar, aunque para facilitar esta tarea
  se puede usar el fichero Makefile incluido.
